Features
========

This module provides the following accounting features specifically designed for Paraguay:

* **VAT Number Validation**: Ensures unique VAT numbers for customers and suppliers
* **Invoice Authorization**: Complete invoice authorization system with proper controls
* **Self-printed Invoices**: Support for self-printed invoice functionality
* **VAT Purchase Book**: Excel report generation for VAT purchases
* **VAT Sale Book**: Excel report generation for VAT sales
* **Book Registration**: Comprehensive book registration system
* **Currency Management**: Enhanced currency handling for Paraguay
* **Email Templates**: Predefined email templates for Paraguay compliance
* **Automated Services**: Cron jobs for automated accounting processes

Models
======

The module extends and creates the following models:

* **Account Move**: Enhanced with Paraguay-specific invoice features
* **Invoice Authorization**: Management of invoice authorization process
* **Book Registration**: VAT book registration and reporting
* **Partner (res.partner)**: Enhanced with Paraguay-specific partner information
* **Currency (res.currency)**: Paraguay-specific currency configurations
* **Company (res.company)**: Company-specific Paraguay settings

Reports
=======

* **VAT Purchase Book**: Comprehensive Excel report for purchase VAT tracking
* **VAT Sale Book**: Comprehensive Excel report for sale VAT tracking
* **Book Registration Reports**: Various reports for compliance and auditing

Installation
============

To install this module, you need to:

#. Clone the repository
#. Add the module to your addons path
#. Update the module list
#. Install the module from Apps menu

Configuration
=============

After installation:

#. Go to Settings > General Settings > Paraguay Localization
#. Configure the invoice authorization settings
#. Set up VAT book registration parameters
#. Configure email templates as needed
#. Set up automated services and cron jobs

Usage
=====

**Invoice Authorization**

#. Create invoice authorizations in Accounting > Configuration > Invoice Authorization
#. Configure authorization parameters and validation rules
#. Assign authorizations to journals and invoice types

**VAT Books**

#. Generate VAT Purchase Book: Accounting > Reporting > VAT Purchase Book
#. Generate VAT Sale Book: Accounting > Reporting > VAT Sale Book
#. Configure book registration parameters as needed

**Self-printed Invoices**

#. Configure self-printed invoice settings in journal configuration
#. Set up invoice numbering sequences
#. Configure invoice templates and layouts
