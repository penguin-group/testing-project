What it does

  • Hides email and password input fields from the login form
  • Removes "Forgot Password" and "Sign Up" links
  • Redirects signup and password reset pages to OAuth providers
  • Maintains all OAuth provider functionality
  • Preserves error and success message display

Before Installation:

  1. Configure at least one OAuth provider in Settings > General Settings > Integrations
  2. Test the OAuth authentication flow
  3. Ensure you have admin access through OAuth

Compatible OAuth Providers

  • Google OAuth
  • Microsoft Azure AD
  • Facebook Login
  • GitHub OAuth
  • LinkedIn OAuth
  • Any custom OAuth 2.0 provider

Technical Details

This module uses Odoo's template inheritance system to override the following templates:

  • web.login - Main login form
  • auth_signup.login - Signup links
  • auth_signup.signup - Signup form
  • auth_signup.reset_password - Password reset form
