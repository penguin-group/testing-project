# Base Tier Validation HR

This module extends the functionality of the tier validation framework to support Human Resources (HR) specific validations.

## Features

- Adds HR-specific tier validation capabilities
- Integrates with Odoo's HR module for employee-based approvals
- Allows defining tier validations based on HR hierarchies and departments
- Extends tier definitions with HR-specific conditions and rules

## Usage

### Configuration

1. Go to Settings > Technical > Tier Validations > Tier Definitions
2. Create a new tier definition for your HR-related model
3. In the 'Validated By' field select 'Job Position'
4. Fill in all other fields as usual.

### Creating Tier Reviews

The module automatically manages tier reviews based on the users associated with the Job Position selected when documents are submitted for validation.
